Sensor Troubleshooting

Android Chrome:
Tap ⋮ → Site settings → Motion sensors → Allow

Samsung Internet:
Settings → Sites and downloads → Motion and orientation → On

iPhone Safari:
Settings → Safari → Motion & Orientation Access → On

All browsers:
Must be on HTTP (not HTTPS) for local network access. The Streamlit URL starting with http://192.168.x.x:8501 is correct.
